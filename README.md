# vue_project
基于 Vue 3 + Vite 的前端项目快速开发模板

## 推荐配置
### 1. 开发IDE
- 工具：[VS Code](https://code.visualstudio.com/)
- 必备插件：Vue (Official)（需禁用 Vetur 插件，避免冲突）

### 2. 浏览器调试
- Chromium内核（Chrome/Edge/Brave）：安装 [Vue.js devtools](https://chromewebstore.google.com/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd)，开启自定义对象格式化
- Firefox：安装 [Vue.js devtools](https://addons.mozilla.org/en-US/firefox/addon/vue-js-devtools/)，开启自定义对象格式化

## 配置自定义
参考 [Vite 官方配置文档](https://vite.dev/config/)，可自定义端口、代理、打包路径等。

## 核心操作命令

# 初始化下载所有依赖包
`npm install` 
# 热更新编译，启动本地服务
`npm run dev`  
# 压缩编译，生成部署文件
`npm run build`
